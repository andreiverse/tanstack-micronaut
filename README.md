# tanstack-micronaut
tanstack & micronaut template with automatic openapi type generation using maven, running with java 25

## Development

**Option 5: Separate Terminals**
```bash
# Terminal 1 - Backend
mvn mn:run

# Terminal 2 - Client
cd client && pnpm run dev
```

## Tech Stack

### Backend
- maven
- micronaut
- flyway
- openapi
- hibernate

### Frontend
- pnpm
- react
- shadcn
- openapi-typescript
- tanstack/start

### Database
- postgresql
